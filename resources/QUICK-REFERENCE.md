## üìù TIPS FOR CALLING

1. **Have ready:**
   - Photo ID
   - Proof of residency (utility bill, lease)
   - Income documentation
   - Social Security numbers (for household members)

2. **Best practices:**
   - Call ahead to confirm hours
   - Ask about required documents
   - Write down reference numbers
   - Save confirmation emails

3. **If you get lost:**
   - Call 2-1-1 for navigation help
   - Ask for a supervisor if needed
   - Local libraries can help with paperwork
